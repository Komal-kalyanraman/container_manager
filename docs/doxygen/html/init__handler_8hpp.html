Detailed Description

Declares project-wide initialization functions for the Container Manager application.

Provides initialization routines for logging, database, message queue, and MQTT subsystems. All functions are designed to be called at application startup to ensure a clean and ready state. The database initialization uses dependency injection for flexibility and testability.
