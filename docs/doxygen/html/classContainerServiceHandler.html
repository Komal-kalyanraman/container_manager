Detailed Description

Handles container operations and updates status in the database.

Provides methods for handling container requests, dispatching to the appropriate command, and updating the status of containers in the database. Uses dependency injection for the database handler to improve testability and flexibility.

Handles a container request by dispatching to the appropriate operation and updating the database. Updates the container status in the database based on the operation result.

Parameters
reqThe container request containing operation, runtime, and container details.
Returns
A JSON object with the result of the operation, including status and error message if any.

Steps:

  1. Perform the requested container operation.
  2. Update the container status in the database based on the operation and result.
  3. Return a JSON object indicating success or error, with operation and container details.
