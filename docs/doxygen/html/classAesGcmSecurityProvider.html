Detailed Description

AES-256-GCM implementation for secure OTA payloads.

Provides authenticated decryption using AES-256-GCM. Loads the key from a hex-encoded file at runtime using AesGcmConfig constants. Expected input format: [12-byte IV][16-byte tag][ciphertext]
