Detailed Description

Handles receiving and processing JSON requests over D-Bus.

This class connects to the D-Bus session bus, registers an object and interface, and listens for incoming method calls (e.g., "Execute") from other applications. Incoming JSON payloads are forwarded to the shared RequestExecutor for processing.
