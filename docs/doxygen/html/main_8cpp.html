Detailed Description

Entry point for the Container Manager application.

This file initializes all core subsystems (logging, database, message queue, MQTT, etc.) and starts all enabled protocol consumers:

  • HTTP server (REST API)
  • MQTT subscriber
  • POSIX Message Queue consumer
  • D-Bus consumer (session bus)

Protocols and data formats are enabled/disabled via CMake flags.
