Detailed Description

Entry point for the Container Manager application.

This file initializes logging, clears the database, and starts all protocol consumers:

  • HTTP server (REST API)
  • MQTT subscriber
  • POSIX Message Queue consumer
  • D-Bus consumer (session bus, for user applications)

All protocols use a shared request executor for processing incoming JSON requests. The application waits for the HTTP server thread to finish and then performs a clean shutdown.
