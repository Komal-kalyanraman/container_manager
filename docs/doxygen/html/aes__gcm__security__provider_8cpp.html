Detailed Description

Implementation of the AesGcmSecurityProvider class for AES-256-GCM decryption.

This file provides authenticated decryption using the AES-256-GCM algorithm. It is used by the executor layer to transparently decrypt incoming encrypted payloads (JSON or Protobuf) for all protocols.
