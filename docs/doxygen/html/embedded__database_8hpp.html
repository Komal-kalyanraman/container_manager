Detailed Description

Declares the EmbeddedDatabaseHandler class for a lightweight, fixed-size in-memory database.

This class provides an embedded-friendly implementation of the IDatabaseHandler interface. It uses a fixed-size array to store key-value pairs, where keys are fixed-length character arrays and values are nlohmann::json objects. This design avoids dynamic memory allocation for keys and limits the number of entries, making it suitable for resource-constrained embedded systems.
