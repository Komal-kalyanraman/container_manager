Detailed Description

Declares command classes for managing containers via the Podman HTTP API over a Unix socket.

Each command class encapsulates a specific Podman API operation (e.g., check runtime, create, start, stop container). These classes use CurlHandler to send HTTP requests to the Podman REST API, using parameters from the ContainerRequest structure.
